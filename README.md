# superapi-tui

TUI application for browsing and interacting with OpenAPI v3.0/v3.1 specifications. Built with Bun, Ink (React for CLI), and TypeScript.

## Install & Run

```bash
# Install dependencies
bun install

# Run in development
bun run dev

# Build for distribution
bun run build

# Run compiled output
node dist/cli.js
```

## Scripts

| Command | Description |
|---------|-------------|
| `bun run dev` | Run in development mode |
| `bun run build` | Build for distribution |
| `bun test` | Run tests |
| `bun run typecheck` | Type checking |
| `bun run lint` | Lint source files |
| `bun run format` | Format source files |
